# Aplikacja Kalendarz

Projekt zaliczeniowy temat nr 1. Aplikacja służy do zarządzania czasem, umożliwiając dodawanie zadań do kalendarza oraz edycję profilu użytkownika.

## Link do wersji live

[https://frontendlaboratoryapp-d445e.web.app](https://frontendlaboratoryapp-d445e.web.app)

## Technologie

- **Frontend**: Next.js 15, React
- **Styling**: Tailwind CSS, Preline UI
- **Backend/Baza danych**: Firebase (Authentication, Firestore, Hosting)
- **Testy**: Playwright

## Funkcjonalności

- Rejestracja i logowanie (Firebase Auth)
- Weryfikacja adresu email
- Zarządzanie profilem użytkownika (zdjęcie, adres)
- Kalendarz z możliwością dodawania, edycji i usuwania zadań
- Responsywny design (RWD) i Tryb Ciemny (Dark Mode)
